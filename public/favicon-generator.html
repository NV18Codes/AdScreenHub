Favicon Generator for AdScreenHub

This tool will generate properly sized favicons from your logo-2.png file.

16x16


32x32


48x48


180x180 (Apple)


Instructions:

  1. Click each "Download" button to save the favicon files
  2. Save them in your public/ folder with these names:
    • favicon-16x16.png (16x16 pixels)
    • favicon-32x32.png (32x32 pixels)
    • favicon-48x48.png (48x48 pixels)
    • apple-touch-icon.png (180x180 pixels)
  3. Also create a favicon.ico file (you can use online converters)
  4. Refresh your browser to see the new favicons!
